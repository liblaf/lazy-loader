## 📦 Installation

To install `liblaf-lazy-loader`, run the following command:

```bash
uv add liblaf-lazy-loader
```

## ⌨️ Local Development

You can use Github Codespaces for online development:

[![](https://github.com/codespaces/badge.svg)](https://codespaces.new/liblaf/lazy-loader)

Or clone it for local development:

```bash
gh repo clone liblaf/lazy-loader
cd lazy-loader
mise run install
```
